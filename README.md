# purescript-quill

(WIP) PureScript bindings for [quilljs](https://quilljs.com/).

## Installation

```
bower install purescript-quill
```

## Example

Build it with:

```bash
$ bower install
$ npm run build:example
```

Then open `example/dist/index.html` in your browser.

```bash
$ open example/dist/index.html
```
